# Contributing to open-claude-skills

This hub only indexes category repositories. Add or edit skills in the matching `skills-*` repo — not here.

Maintainer: **Poorvith M P** · v0.2 · July 2026

See each category repo's `CONTRIBUTING.md` for skill-creator rules.

## Adding a new skill

1. Go to the matching category repo (e.g. [skills-developer](https://github.com/prvthmpcypher/skills-developer))
2. Follow that repo's `CONTRIBUTING.md`
3. Once merged there, open a PR here to add the skill to the index table in `README.md`
